# Templates 
## Λογική  
Έστω ότι θέλω να προσθέσω διαφορετικούς τύπους δεδομένων πχ ακεραίους, float, strings 
Μπορώ να δημιουργήσω μια πρότυπη συνάρτηση που να είναι ανεξάρτητη από τον τύπο των ορισμάτων και τον τύπο επιστροφής.  
Αυτό είναι template.  
*θέλω ένα "καλούπι" για να κάνω πχ προσθεση ή πχ για να κάνω sort διαφορετικά αντικείμενα*
## Υλοποίηση
Υπάρχει γι' αυτό η δεσμευμένη λέξη template.  
Τοτε υλοποιώ τη συνάρτηση ως template και μετά όταν τη χρησιμοποιώ δηλώνω τον τύπο.  
```cpp 
template <class X> void mySwap(X &a, X &b)
{
    X temp; 
    temp = a;
    a = b;
    b = temp;
}
int main(void)
{
    int x = 10, y =20;
    double a = 1121.121, b = 1212.3;
    mySwap(x, y);
    mySwap(a, b);
}
```
### *Παρατηρήσεις:*
1. Το wildcard το δηλώνω με την παράμετρο **<class X>**.  
2. Ορίζουμε δηλαδή πράγματα τα οποία αλλάζουν κατά τη μεταγλώττιση. Δηλαδή ο compiler κάνει τη δουλειά.
3. Ομοίως μπορώ να επιλέγω και τον τύπο επιστροφής.  
```cpp 
template <class type1, class type2>
    void myFunc()
    {
        cout << ...
    }
```
## Ρητή υπερφόρτωση της βασικής συνάρτησης. 
```cpp 
template <class X> void mySwap(X &a, X&b);
void mySwap(int &a, int &b);
```
Αλλά κανονικά:
```cpp 
template <class X> void mySwap(X &a, X&b);
template<> void mySwap<int>(int &a, int &b); // Right way!!!!!!!
```
Έτσι κάναμε ένα exemption.  
## Υπερφόρτωση γενικής συνάρτησης
```cpp 
template <class X, class Y> void function(X &a, Y &b)
{
}
```
## Γενικές κλάσεις 
```cpp 
template<class type1> class class
{
    type1 i;
public:
    Myclass(type1 i)
    {
        i = a;
    }
    void show();
}
template <class type1>
    void Myclass<type1>::show()
    {
        cout << i << endl;
    }
```
## Ατυπα ορίσματα 
Ποιοι είναι οι 3 τύποι άτυπων ορισμάτων:
1. int  
2. references  
3. pointers  
Αυτό το κάνω μόνο αν έχω pointer και συνήθως θέλω να διαχειριστώ πίνακα.  
## Predefined 
```cpp 
template <class type1 = int, int size = 2> class Myclass
{
}
```

->Με τον ίδιο τρόπο που γίνεται ρητή εξειδίκευση συνάρτησης.  
Exemption: 
```cpp 
template<> class Myclass<int>

```
