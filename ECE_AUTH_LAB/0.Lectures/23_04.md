## Στατικά μέλη κλασης
Στατικό -> Για όσα αντικέίμενα δημιουργηθούν από ένα αντικείμενο/τύπο κλάσης, το στοιχείο/μεταβλητή είναι κοινή για όλα τα στιγμιότυπα. 
πχ Αν αλλάξω μία τιμή τότε όλα τα αντικείμενα θα έχουν ως τιμή τη νέα μεταβλητή.
Άρα το αντικείμενο δεν θα αρχικοποιήσει μία τιμή απλά όλα τα αντικείμενα κάνουν reference στην ίδια θέση μνήμης.
Όταν σε μία κλάση ορίζεται static **ΔΕΝ ΜΠΟΡΩ ΝΑ ΑΡΧΙΚΟΠΟΙΗΣΩ ΣΤΟΝ CONSTRUCTOR**. 
e.g. 
```
class SharedVar{
public:
    static int num;
};
int SharedVar::num = 4;
int main()
{
    SharedVar a, b;
    cout << a.num << " " << b.num;
}
```  
2 Τρόποι προσπέλασης:
**Παρατηρώ ότι τη δήλωσα public**  
Για να την προστατέψω απλά κάνω Setters and getters αλλά **πρέπει να είναι και αυτοί static**.  
```
class SharedVar{
public:
    static void show();
};
int SharedVar::show()
{
    cout << "BLA BLA" << endl;
}
int main()
{
    SharedVar a, b;
    cout << a.num << " " << b.num;
}
```
Στην πραγματικότητα χρησιμοποιώ static μεθόδους για να κάνω set σε static μεταβλητές.  
Προσοχή πρέπει πάντα να αρχικοποιώ τις τιμές.  
Αν βάλω Static συνάρτηση να προσπελαύνει non static μεταβλητή θα βγάλει προφανώς error.
**Θα είχε νόημα να έχω συνάρτηση με static ορίσματα?**
ΟΧΙ!!!Απλά θα βάλω default arguments:
`
e.g. int num(int k = 100);
`

## Δείκτες σε βασικούς τύπους(pointers to base classes)
e.g.
```
Bclass *p;
Bclass bobj;
Dclass dobj;

p = &bobj; //Όταν θα μιλήσω σε αυτό το αντικείμενο θα καταλάβει ότι το αντικείμενο είναι τύπου base class.
p = &dobj; // Μπορώ να έχω pointer στη βασική κλάση και να προσπελεύνω όλα τα αντικείμενα σε οποιαδήποτε πααραγόμενη κλάση 
// Dclass is publicly derived from dclass.
```
Άρα δεν χρειάζομαι Typecasting!!!!!!!!(with this procedure)
Μπορώ να μιλήσω σε όλα τα αντικείμενα με μόνο 1 pointer στη baseclass.
Δηλαδή με έναν pointer μπορώ να προσπελεύνω και παραγόμενες κλάσεις.
**Προσοχή!!!!** ΔΕΝ ΜΠΟΡΩ ΝΑ ΧΡΗΣΙΜΟΠΟΙΗΣΩ POINTER ΣΕ ΠΑΡΑΓΟΥΣΑ ΚΛΑΣΗ ΓΙΑ ΠΡΟΣΠΕΛΑΣΗ ΜΕΘΟΔΩΝ ΠΑΡΑΓΟΜΕΝΗΣ.
Θέλω λοιπόν έναν αυτοματοποιημένο τρόπο ώστε όλες οι παραγόμενες κλάσεις να επικοινωνούν με την αρχική.  
Θέλω λοιπόν να γλιτώσω το Typecasting.
ΑΝ έκανα Typecasting θα έπρεπε ΚΑΘΕ ΦΟΡΑ ΠΟΥ ΚΑΛΩ ΚΑΤΙ να κάνω Typecasting...Παντού και στην ανάθεση και στην κλήση κλπ.

### Εικονικές Βασικές κλάσεις
Μια εικονική κλάση μας λέει ότι όταν θα πας να υλοποιήσεις την κλάση, όταν θα πας να υλοπιήσεις την κλάση derived από 2 άλλες παραγόμενες μην έχεις 2 Baseclass μεταβλητές αλλά 1.  
π.χ.
```
class Baseclass{
public:
int i;
};
class Derived2 : public Baseclass
{
public:
int k;
};
class Derived3 : public Baseclass
{
public:
int l;
};
class Derived1 : public Derived2, public Derived3 
{
};
```
